# OpenPassportSwift (iOS Only)

Noir passport ZK circuits + mopro bindings packaged in a `SemaphoreSwift`-style Swift Package for iOS.

## Repository Format

```text
circuits/                    # Noir circuits
mopro-binding/               # Rust FFI + mopro iOS build entry
Package.swift                # Swift Package manifest
Sources/
  OpenPassportSwift.swift    # Swift package entry point
  MoproiOSBindings/          # Generated by mopro (mopro.swift + xcframework)
Tests/
  OpenPassportSwiftTests.swift
```

## iOS-Only Scope

- Supported: iOS bindings and Swift Package integration.
- Not included: Android bindings.

## Build iOS Bindings

```bash
# 1) Compile and test circuits
make circuits

# 2) Build mopro iOS bindings and sync them into Sources/MoproiOSBindings
make build-ios

# Optional: include Intel simulator slice too
# IOS_ARCHS="aarch64-apple-ios,aarch64-apple-ios-sim,x86_64-apple-ios" make build-ios
```

`make build-ios` does:
1. Copy compiled circuit JSON files to `mopro-binding/test-vectors/noir/`
2. Run `IOS_ARCHS=<...> IPHONEOS_DEPLOYMENT_TARGET=15.0 CONFIGURATION=release cargo run --bin ios`
3. Sync `mopro-binding/MoproiOSBindings` to `Sources/MoproiOSBindings`

## Use as Swift Package

### Option 1: Xcode

1. In Xcode: `File` → `Add Package Dependencies`
2. Enter this repo URL
3. Add product: `OpenPassportSwift`

### Option 2: `Package.swift`

```swift
dependencies: [
    .package(url: "https://github.com/kidneyweakx/conductor-playground.git")
],
targets: [
    .target(
        name: "YourApp",
        dependencies: ["OpenPassportSwift"]
    )
]
```

### Option 3: CocoaPods

```ruby
pod 'OpenPassportSwift', :git => 'https://github.com/kidneyweakx/conductor-playground.git'
```

## Swift Usage

```swift
import OpenPassportSwift

let circuitPath = Bundle.main.path(forResource: "disclosure", ofType: "json")!

let inputs: [String: [String]] = [
    "mrz_data": mrzFieldValues,
    "mrz_hash": mrzHashValues,
    "disclose_nationality": ["1"],
    "disclose_older_than": ["0"],
    "disclose_name": ["0"],
    "age_threshold": ["18"],
    "current_date": currentDateYYMMDD,
    "out_nationality": outNationality,
    "out_name": outName,
    "out_is_older": ["0"]
]

let result = try generateNoirProof(circuitPath: circuitPath, srsPath: nil, inputs: inputs)
let isValid = try verifyNoirProof(proof: result.proof, vk: result.vk)
```
